# Theorem Proving in Lean 4

This directory contains copied examples, answers to exercises, and notes created while reading [_Theorem Proving in Lean 4_](https://lean-lang.org/theorem_proving_in_lean4/) by Jeremy Avigad, Leonardo de Moura, Soonho Kong, and Sebastian Ullrich, with contributions from the Lean Community.

The folders and files in this directory correspond to sections of the book.
